# 数字孪生系统项目说明文档

## 项目概述

数字孪生系统是一个基于Web的3D可视化平台，结合了现代前端技术和后端服务，为用户提供沉浸式的数字孪生体验。系统支持3D场景编辑、模型管理、贴图应用等功能，适用于工业制造、智慧城市、建筑信息建模等多个领域。

## 技术架构

### 前端技术栈

- **Vue.js 3.x** - 渐进式JavaScript框架，用于构建用户界面
- **Element Plus** - Vue 3组件库，提供丰富的UI组件
- **Three.js** - 3D图形库，实现WebGL渲染和交互
- **Vite** - 新一代前端构建工具，提供快速的开发体验
- **Axios** - HTTP客户端，用于与后端API通信

### 后端技术栈

- **Spring Boot 2.x** - Java Web框架，简化企业级应用开发
- **MyBatis-Plus** - 数据库访问框架，简化数据库操作
- **MySQL 8.0** - 关系型数据库，存储系统数据
- **JWT** - JSON Web Token，实现用户身份认证

## 核心功能模块

### 1. 3D场景编辑器

- **场景创建与管理**：支持新建、保存、加载和导出3D场景
- **模型操作**：提供模型的添加、删除、位置调整、缩放和旋转
- **交互控制**：通过鼠标实现视角控制、模型选择等操作
- **可视化反馈**：选中模型时提供高亮显示效果

### 2. 模型管理系统

- **模型库浏览**：分类展示可用的3D模型
- **模型搜索**：支持按名称搜索模型
- **组合模型**：支持创建和管理由多个模型组成的复杂结构
- **模型预览**：提供模型的快速预览功能

### 3. 贴图管理系统

- **多文件支持**：支持同时选择多个贴图文件
- **贴图类型**：支持基础颜色、法线、粗糙度、金属度等多种贴图类型
- **可视化管理**：文件列表显示，支持单独删除文件
- **实时预览**：提供贴图预览功能，确保应用效果

### 4. 用户权限系统

- **用户认证**：注册、登录和会话管理
- **权限控制**：基于角色的访问控制机制
- **安全管理**：JWT令牌验证，确保系统安全

### 5. 后台管理系统

- **仪表板**：系统概览和统计信息展示
- **用户管理**：用户信息维护和权限分配
- **设备管理**：设备信息管理和状态监控
- **场景管理**：场景资源的集中管理

## 部署要求

### 前端部署

- Node.js 14+ 环境
- 支持现代浏览器（Chrome 90+, Firefox 88+, Edge 90+, Safari 14+）
- Web服务器（Nginx/Apache）用于生产环境部署

### 后端部署

- Java 8+ 运行环境
- MySQL 8.0+ 数据库
- 至少4GB内存的服务器环境

## 开发指南

### 环境搭建

1. 克隆项目代码到本地
2. 前端：安装Node.js依赖 `npm install`
3. 后端：配置MySQL数据库并初始化表结构
4. 修改配置文件中的数据库连接信息

### 启动项目

- 前端开发：`npm run dev`
- 后端服务：通过IDE运行或打包为jar文件执行
- 访问地址：<http://localhost:指定端口>

## 使用说明

### 快速开始

1. 注册并登录系统
2. 创建新的3D场景
3. 从模型库中添加模型到场景
4. 调整模型位置、大小和方向
5. 为模型应用贴图增强视觉效果
6. 保存场景并在场景库中管理

### 高级功能

- 创建组合模型以复用复杂结构
- 使用不同类型的贴图提升模型真实感
- 通过后台管理系统维护系统数据

## 维护和支持

### 常见问题

- 模型加载失败：检查模型文件格式和路径
- 贴图不显示：确认贴图类型和模型材质兼容性
- 登录异常：检查网络连接和服务器状态

### 技术支持

- 查阅系统日志定位问题
- 参考官方文档和社区资源
- 联系开发团队获取专业支持
