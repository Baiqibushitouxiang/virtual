# 一期接口测试清单

## 1. 使用说明
- 目标：按业务顺序验证一期改造后的接口功能、逻辑闭环与关键兼容点。
- 执行方式：
  - 自动化测试：运行 Maven 单元/控制器测试
  - 联调测试：按本清单逐条执行并填写实际结果
- 环境前置：
  - 本地 MySQL `dam` 可用
  - 已执行 `model_data_binding.sql` 与 `alert_rule.sql`
  - 后端服务已启动
  - 如需验证场景资源上传，准备 `json/glb/gltf` 文件

## 4. 关键验证点
- 场景导出 JSON 再导入后，模型位置、缩放、旋转、天空盒与绑定关系恢复正确。
- 同一 `sceneId + modelId` 重复保存绑定时，应更新而不是产生重复脏数据。
- 展板展示以模型绑定为准，不再要求展板单独选择设备。
- 告警规则支持 `>` `>=` `<` `<=` 四种比较方式。
- 告警通知当前仅控制台打印，但必须能看见规则 ID、设备编码、数据项与当前值。
- 设备注册成功后，即使 OPC UA 节点创建失败，也不应回滚主事务。
- 所有 controller 自动化用例至少断言 `HTTP 200/204` 与统一返回结构中的关键字段。
