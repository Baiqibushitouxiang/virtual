# 数字孪生建模平台用户手册

## 1. 系统概述

数字孪生建模平台是一个基于Web的3D场景建模工具，支持用户创建、编辑和管理数字孪生场景。平台提供直观的界面和丰富的功能，使用户能够轻松构建复杂的3D模型组合和场景。

## 2. 系统架构

### 2.1 前端技术栈
- Vue 3：前端框架
- Element Plus：UI组件库
- Three.js：3D渲染引擎
- Font Awesome：图标库
- Vite：构建工具

### 2.2 后端技术栈
- Spring Boot：后端框架
- Java：开发语言

## 3. 登录与注册

### 3.1 用户注册

1. **访问注册页面**：
   - 在浏览器中输入平台地址，点击"立即注册"链接

2. **填写注册信息**：
   - **用户名**：输入3-20个字符的用户名
   - **邮箱**：输入有效的邮箱地址
   - **密码**：输入6-20个字符的密码
   - **确认密码**：再次输入相同的密码

3. **提交注册**：
   - 点击"立即注册"按钮
   - 注册成功后，系统将自动跳转到登录页面

### 3.2 用户登录

1. **访问登录页面**：
   - 在浏览器中输入平台地址，进入登录页面

2. **输入登录凭证**：
   - **用户名**：输入注册时的用户名
   - **密码**：输入注册时的密码

3. **登录系统**：
   - 点击"登录"按钮
   - 登录成功后，系统将自动跳转到主界面

## 4. 主界面功能

登录成功后，用户将进入三维场景编辑器主界面，界面主要包含以下部分：

### 4.1 顶部导航栏

- **返回主界面**：点击左侧箭头图标返回主界面
- **选择模型**：点击按钮打开模型选择器
- **保存场景**：保存当前编辑的场景
- **撤销**：撤销上一步操作
- **清除场景**：清除当前场景中的所有模型
- **使用组合**：打开组合模型选择器
- **创建组合**：打开组合模型编辑器
- **切换天空背景**：切换场景的天空背景效果
- **导出GLB**：将场景导出为GLB格式
- **导出JSON**：将场景导出为JSON格式

### 4.2 3D场景区域

- 主要的3D模型显示和编辑区域
- 支持鼠标交互：旋转、缩放、平移场景
- 支持选择和编辑模型

### 4.3 模型选择器

1. **打开模型选择器**：
   - 点击工具栏中的"选择模型"按钮

2. **浏览模型**：
   - 在模型选择器中浏览可用的3D模型

3. **选择模型**：
   - 点击模型缩略图选择需要添加的模型
   - 模型将被添加到当前场景中

### 4.4 组合模型选择器

1. **打开组合模型选择器**：
   - 点击工具栏中的"使用组合"按钮

2. **浏览组合模型**：
   - 在选择器中浏览已保存的组合模型

3. **加载组合模型**：
   - 选择需要加载的组合模型
   - 组合模型将被添加到当前场景中

### 4.5 组合模型编辑器

1. **打开组合模型编辑器**：
   - 点击工具栏中的"创建组合"按钮

2. **编辑组合模型**：
   - **模型列表**：显示当前场景中的所有模型
   - **属性编辑**：修改选中模型的属性
   - **组合管理**：添加或删除组合中的模型

3. **保存组合模型**：
   - 点击"保存组合模型"按钮
   - 输入组合模型名称和描述
   - 保存成功后，组合模型将可在选择器中使用

## 5. 场景库管理

### 5.1 访问场景库

- 在左侧菜单栏中点击"场景库"选项

### 5.2 场景列表

- **场景信息**：显示场景的ID、名称、类型、存储路径和创建时间
- **场景操作**：
  - **使用**：将选中的场景加载到编辑器中
  - **删除**：删除选中的场景

### 5.3 上传场景

1. **打开上传对话框**：
   - 点击"上传场景(JSON/GLB/GLTF)"按钮

2. **选择文件**：
   - 选择需要上传的场景文件（支持JSON、GLB、GLTF格式）

3. **上传场景**：
   - 点击上传按钮
   - 上传成功后，场景将显示在场景列表中

### 5.4 刷新列表

- 点击"刷新列表"按钮更新场景列表

## 6. 后台管理

### 6.1 控制面板

- 显示系统的总体统计信息
- 提供系统概览和关键指标

### 6.2 设备管理

- **设备列表**：显示所有设备的基本信息
- **设备操作**：添加、编辑、删除设备

### 6.3 位置信息管理

- **位置列表**：显示所有设备的位置信息
- **位置操作**：添加、编辑、删除位置信息

### 6.4 用户管理

- **用户列表**：显示所有用户的基本信息
- **用户操作**：添加、编辑、删除用户
- **权限管理**：设置用户的权限级别

## 7. 操作指南

### 7.1 基本场景操作

1. **添加模型**：
   - 点击"选择模型"按钮
   - 在模型选择器中选择模型
   - 模型将被添加到场景中

2. **保存场景**：
   - 点击"保存场景"按钮
   - 输入场景名称和描述
   - 场景将被保存到场景库中

3. **导出场景**：
   - 点击"导出GLB"或"导出JSON"按钮
   - 选择保存路径
   - 场景文件将被下载到本地

### 7.2 组合模型操作

1. **创建组合模型**：
   - 在场景中添加多个模型
   - 点击"创建组合"按钮
   - 在组合编辑器中调整模型属性
   - 保存组合模型

2. **使用组合模型**：
   - 点击"使用组合"按钮
   - 在组合选择器中选择需要的组合模型
   - 组合模型将被添加到场景中

### 7.3 场景管理

1. **加载场景**：
   - 进入场景库
   - 选择需要加载的场景
   - 点击"使用"按钮
   - 场景将在编辑器中打开

2. **删除场景**：
   - 进入场景库
   - 选择需要删除的场景
   - 点击"删除"按钮
   - 确认删除操作

## 8. 系统配置与维护

### 8.1 浏览器要求

- Chrome 80+
- Firefox 75+
- Safari 13+
- Edge 80+

### 8.2 系统要求

- 操作系统：Windows 10+, macOS 10.15+, Linux
- 内存：8GB以上
- 显卡：支持WebGL的独立显卡
- 网络：稳定的网络连接

### 8.3 常见问题排查

1. **模型无法加载**：
   - 检查网络连接
   - 确认模型文件格式正确
   - 检查浏览器是否支持WebGL

2. **场景保存失败**：
   - 检查网络连接
   - 确认场景名称和描述填写完整
   - 检查服务器状态

3. **界面显示异常**：
   - 清除浏览器缓存
   - 刷新页面
   - 检查浏览器版本
